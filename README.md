# Java设计模式
> 设计模式(Design Pattern)是一套被反复使用、多数人知晓的、经过分类编目的、代码设计经验的总结，使用设计模式是为了可重用代码、让代码更容易被他人理解并且保证代码可靠性。

## 设计模式可分为
1. 创建型(Creational): 主要用于描述如何创建对象 
2. 结构型(Structural): 主要用于描述如何实现类或对象的组合
3. 行为型(Behavioral): 主要用于描述类或对象怎样交互以及怎样分配职责
